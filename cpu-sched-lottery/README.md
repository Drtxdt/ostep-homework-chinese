# 概述

此程序 lottery.py 可让你观察彩票调度器如何工作。和往常一样，运行程序分为两个步骤。首先，在不使用 -c 标志的情况下运行：这会展示需要解决的问题而不给出答案。

```
提示符> ./lottery.py -j 2 -s 0
...
以下是作业列表，包含每个作业的运行时间：
  作业 0 ( 长度 = 8, 彩票数 = 75 )
  作业 1 ( 长度 = 4, 彩票数 = 25 )

以下是您将需要的随机数集合（最多需要）：
随机数 511275
随机数 404934
随机数 783799
随机数 303313
随机数 476597
随机数 583382
随机数 908113
随机数 504687
随机数 281838
随机数 755804
随机数 618369
随机数 250506
```

当你以这种方式运行模拟器时，它首先分配一些随机作业（此处长度为 8 和 4），每个作业有一定数量的彩票（此处分别为 75 和 25）。模拟器还提供一个随机数列表，你将需要用这些随机数来确定彩票调度器的行为。随机数选择在 0 和一个大数之间；因此，你需要使用取模运算符来计算彩票赢家（即，赢家应等于此随机数对系统中总彩票数的模）。

使用 -c 运行将精确显示你应计算的内容：

```
提示符> ./lottery.py -j 2 -s 0 -c
...
** 解答 **
随机数 511275 -> 中奖彩票 75 (总票数 100) -> 运行 1
  作业： (  作业:0 剩余时间:8 票:75 ) (* 作业:1 剩余时间:4 票:25 )
随机数 404934 -> 中奖彩票 34 (总票数 100) -> 运行 0
  作业： (* 作业:0 剩余时间:8 票:75 ) (  作业:1 剩余时间:3 票:25 )
随机数 783799 -> 中奖彩票 99 (总票数 100) -> 运行 1
  作业： (  作业:0 剩余时间:7 票:75 ) (* 作业:1 剩余时间:3 票:25 )
随机数 303313 -> 中奖彩票 13 (总票数 100) -> 运行 0
  作业： (* 作业:0 剩余时间:7 票:75 ) (  作业:1 剩余时间:2 票:25 )

  作业： (  作业:0 剩余时间:6 票:75 ) (* 作业:1 剩余时间:1 票:25 )
--> 作业 1 于时间 6 完成
随机数 908113 -> 中奖彩票 13 (总票数 75) -> 运行 0
  作业： (* 作业:0 剩余时间:6 票:75 ) (  作业:1 剩余时间:0 票:--- )
随机数 504687 -> 中奖彩票 12 (总票数 75) -> 运行 0
  作业： (* 作业:0 剩余时间:5 票:75 ) (  作业:1 剩余时间:0 票:--- )

  作业： (* 作业:0 剩余时间:3 票:75 ) (  作业:1 剩余时间:0 票:--- )

  作业： (* 作业:0 剩余时间:2 票:75 ) (  作业:1 剩余时间:0 票:--- )
```

从此跟踪中可以看出，你应该做的是使用随机数找出哪张彩票是赢家。然后，根据中奖彩票，确定应该运行哪个作业。重复此过程，直到所有作业运行完毕。就这么简单——你只是在手动模拟彩票调度器的行为！

为了完全明确起见，我们来看一下上面示例中的第一个决策。此时，我们有两个作业（作业 0 运行时间为 8，拥有 75 张彩票；作业 1 运行时间为 4，拥有 25 张彩票）。我们得到的第一个随机数是 511275。由于系统中有 100 张彩票，511275 % 100 等于 75，因此 75 是我们的中奖彩票。

如果第 75 号彩票是赢家，我们只需遍历作业列表，直到找到它。第一个条目是作业 0，拥有 75 张彩票（从 0 到 74），因此未中奖；下一个条目是作业 1，于是我们找到了赢家，所以我们运行作业 1 一个时间量子长度（在此示例中为 1）。所有这些都在打印输出中显示如下：

```
随机数 511275 -> 中奖彩票 75 (总票数 100) -> 运行 1
  作业： (  作业:0 剩余时间:8 票:75 ) (* 作业:1 剩余时间:4 票:25 )
```

如你所见，第一行总结了发生的情况，第二行仅显示了整个作业队列，并用 * 表示哪个作业被选中。

模拟器还有一些其他选项，其中大多数应能一目了然。最值得注意的是，-l/--jlist 标志可用于指定一组确切的作业及其彩票值，而不是总是使用随机生成的作业列表。

```
提示符> ./lottery.py -h
用法: lottery.py [选项]

选项:
  -h, --help            
      显示此帮助信息并退出
  -s SEED, --seed=SEED  
      随机种子
  -j JOBS, --jobs=JOBS  
      系统中的作业数量
  -l JLIST, --jlist=JLIST
      不随机生成作业，而是提供一个以逗号分隔的
      运行时间和彩票值列表（例如，10:100,20:100
      表示两个作业，运行时间分别为 10 和 20，
      每个作业拥有 100 张彩票）
  -m MAXLEN, --maxlen=MAXLEN
      作业的最大长度
  -T MAXTICKET, --maxtick=MAXTICKET
      如果随机分配，彩票的最大值
  -q QUANTUM, --quantum=QUANTUM
      时间片长度
  -c, --compute
      为我计算答案
```
